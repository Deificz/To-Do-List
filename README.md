# To-Do-List

A web based To-Do-List wherein you can list your upcoming tasks or projects.

Usage:
- To add task click the Add Task button
- To add Project or Plans click the Plus Sign Button

Tech stack used:
  -  HTML
  -  CSS
  -  Vanilla JavaScript
  -  localStorage
  -  Webpack
